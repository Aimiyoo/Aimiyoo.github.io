创建分支

  1. fetch
    相当于从远程获取最新版本到本地，不会自动merge

  2. checkout
    获取远程分支remoteName到本地新分支localName，并跳到localName分支

    pull相当于git fetch和git merge

  1. 获取远程tag

  3. 删除远程tag

    或者

stash

  1. 添加stash
    git stash用于保存当前的工作进度，不用commit即可checkout到另外的branch

  2. 显示stash列表
    git stash list用于显示所有的stash列表

  3. 恢复stash
    git stash pop用于恢复某一次的改动，默认不加hash代表恢复最近的一次改动

  4. 删除stash
    git stash drop用于删除某一次的改动，默认不加hash代表删除最近的一次改动

  5. git stash pop和git stash apply区别
    pop命令从stash列表中恢复并删除某一次改动
    apply命令从stash恢复但不删除某一次改动
