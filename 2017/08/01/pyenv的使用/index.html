pyenv的使用

  • python

Introduction

pyenv lets you easily switch between multiple versions of Python. It’s simple, unobtrusive, and follows the UNIX tradition of single-purpose tools that do one thing well.

Installation

  1. check out pyenv where you want it installed

  2. define environment

Usage

  1. view available python versions

  2. install python

  3. select the vesion which you want to use
    now you have two python versions in your system, you can select the version which you want to use

Note

  1. pyenv global {version} not work
    solution: 
    $ pyenv local --unset

Reference

https://github.com/pyenv/pyenv
