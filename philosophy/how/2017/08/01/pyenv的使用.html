pyenv，神器神器

Introduction

pyenv lets you easily switch between multiple versions of Python. It’s simple, unobtrusive, and follows the UNIX tradition of single-purpose tools that do one thing well.

Installation

  1. check out pyenv where you want it installed

     git clone https://github.com/pyenv/pyenv.git ~/.pyenv

  2. define environment

     echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bashrc
 echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bashrc
 echo 'eval "$(pyenv init -)"' >> ~/.bashrc
 source ~/.bashrc

Usage

  1. view available python versions

     pyenv install --list

  2. install python

     pyenv install 2.7.13 -v
 pyevn install 3.6.2 -v
 pyenv rehash

  3. select the vesion which you want to use

    now you have two python versions in your system, you can select the version which you want to use

     pyenv global 2.7.9

change pip source

mkdir ~/.pip
vim ~/.pip/pip.conf

# content
[global]
timeout = 60000
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
[install]
use-mirrors = true
mirrors = https://pypi.tuna.tsinghua.edu.cn

Note

  1. pyenv global {version} not work

     pyenv local --unset

  2. common build problem 

     https://github.com/pyenv/pyenv/wiki/common-build-problems
